Properties Reference

DuxForm

Property Default Description
name (required) Name of form
onSubmit (none) Called when the form is submitted. See Getting Started for an example.
onValidate (none) Called to validate the form. See Validation for more information.
initialFocus (none) The name of a form field to receive initial focus when the form initializes.

DuxInput

Property Default Description
name (required) Name of form field
nextField (none) The name of the form field to receive focus when the user pressed Enter
onValidate (none) Called to validate the value. See Validation for more information.
onEnterPressed (none) Called when the user presses Enter. The function is passed the current value, the field name, and the form name.
format (none) Called to format the value. See Formatting for more information.
parse (none) Called to parse the value. See Formatting for more information.
normalize (none) Called to normalize the value. See Formatting for more information.
dataType str The data type of the field. See Data Types for more information.
precision 0 The precision of the number field. See Data Types for more information.
showCommas false Whether to use commas as a thousands separator. See Data Types for more information.
showDollar false Whether to show a dollar sign in the field. See Data Types for more information.
round true Whether to the value based on the precision. See Data Types for more information.
forceUpper false Whether to force letters to upper case. See Data Types for more information.
trim true Whether to remove leading and trailing whitespace when the field loses focus.
onFocus (none) Called when the field gains focus. The function is passed the field name.
onBlur (none) Called when the field loses focus. The function is passed the field name.

DuxAutoComplete

Property Default Description
name (required) Name of form field
items (required) The items the user can select from. See AutoComplete Fields for more information.
allowMulti false True of the user is allowed to select multiple items in the list
multiSeparators [',', ';'] Array of characters that users type to begin a new item in a multi-select list
allowNewItems false True if the user is allowed to add items not in the supplied list of items
validateItem (none) Callback function to validate items before adding to a multi-select list
