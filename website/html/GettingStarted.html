Getting Started

If you're the ready, fire, aim type (aren't all programmers?), you should find everything you need on this page to start using DuxForm.

Installation

    npm install duxform

If you are using the <DuxAutoComplete> component, you also need to link to the duxautocomplete.css file in the dist/ directory. 

    <link rel="stylesheet" href="duxautocomplete.css"/>

Redux Store

DuxForm stores form state in the Redux store. It requires the root form object in the store. A reducer is supplied to manage this. 

    import {DuxFormReducer} from 'duxform';

    const appReducer = combineReducers({
        ...  // other reducers
        forms: DuxFormReducer
    });

Browser Requirements

DuxForm doesn't place any special requirements on browser support. It has been tested on the modern evergreen browsers (Chrome and Firefox) as well as Internet Explorer 10 and 11. I know for a fact it won't work on Internet Explorer 8 or earlier. The <DuxAutoComplete> component uses Font Awesome icons which require Internet Explorer 10 or 11.
