Input Types

<DuxInput> supports a few <input> type attributes natively. Other types such as password and number work as well but do not include any built-in support. Five basic input types are supported - text, checkbox, radio, select and textarea. The last two are special cases. The text type, like the <input> tag, is the default. Nothing special is done when the type is text.

Checkbox

Checkboxes store their value in the Redux store as a JavaScript boolean value. Like the React <input> tag, use the defaultChecked property to specify a default value for the checkbox. The default is false (unchecked).

Radio Buttons

A group of radio buttons with the same name property will share a single value in the Redux store. The value in the store reflects the currently selected button in the group. Use the defaultChecked property to specify a radio button to be selected by default. Following is an example. 

    <DuxForm>
        <DuxInput name="choice" value="first" type="radio" defaultChecked={true}/>
        <DuxInput name="choice" value="second" type="radio"/>
        <DuxInput name="choide" value="third" type="radio"/>
    </DuxForm>

In this example, the Redux store will have just one field named choice which will contain a value of first, second, or third based on the user selection.

Select Tags

Setting type="select" will render a <select> tag in the document. Use the defaultValue property to specify the option selected by default. The value in the Redux store for the field will be the value of the currently selected option. Following is an example. 

    <DuxInput name="color" defaultValue="blue" type="select">
        <option value="red">Red</option>
        <option value="green">Green</option>
        <option value="blue">Blue</option>
        <option value="yellow">Yellow</option>
    </DuxInput>

TextArea

Setting type="textarea" will render a <textarea> tag in the document. Use the defaultValue property to specify the default value in the field. This is different from the React <textarea> tag in that, the default value is a child of the tag.
